What is the difference between Margin, Border, and Padding?

Take me back!

In CSS, a margin is the outer layer of the screen, which is then followed by the border, the padding and then the content inside.




Here we have a box made from only A margin.

As you can see there is nothing there as the margin is a transparent invisible border which helps you make an invisible area if you needed to cut out areas for nothing to go in.

The code for my margin is:
.margin {
width: 100px;
height: 100px;
margin: 6px solid rgb(23, 160, 188);
}



Here we have a box made only using a border.

The borders color be easily be manipulated to whatever color you like, borders could be use for many things, like an entire page border or small little boxes.

The code for my border is:
.border {
width: 100px;
height: 100px;
border: 6px solid rgb(23, 160, 188);
}



Here is a box made only from padding.

Here the padding is actually invisible and only the background can be manipulated in its color, could be good for an small area to have its own backround!

The code for my padding is:
.padding {
width: 100px;
height: 100px;
padding: 6px;
background: rgb(23, 160, 188);
}
